Practicas HTML

HTML es un leguaje de etiquetas, Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language es el componente más básico de la Web. Define el significado y la estructura del contenido web. HTML es un componente esencial en el desarrollo de páginas web, pero es importante recordar que no es un lenguaje de programación en sí mismo. Para agregar funcionalidad dinámica y estilos a una página web, se utilizan otros lenguajes como CSS y JavaScript.

  • La etiqueta !DOCTYPE html> indica al navegador que el documento que se está cargando es un documento HTML.
  • La etiqueta html> es el contenedor principal de todo el documento HTML.
  • La etiqueta head> contiene información sobre el documento, como el título de la página y la codificación de caracteres.
  • La etiqueta meta charset="UTF-8"> especifica que el documento utiliza la codificación de caracteres UTF-8.
  • La etiqueta title> define el título de la página web que aparecerá en la pestaña del navegador.
  • La etiqueta body> contiene todo el contenido visible de la página web.
  • La etiqueta h1> define un encabezado de nivel 1 (título principal).
  • La etiqueta p> define un párrafo.
  • La etiqueta ul> define una lista no ordenada.
  • La etiqueta li> define un elemento de la lista.
  • La etiqueta img> inserta una imagen en la página web, utilizando la propiedad src para especificar la URL de la imagen y la propiedad alt para proporcionar una descripción alternativa.
